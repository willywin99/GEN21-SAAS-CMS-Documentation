Schedule For Recording

Schedule For Recording 

Description

To create a schedule for recording and it will send the reminder to the users to record the live event/program.No integration with any other external application, such as: Media Client, etc

Application Configuration

Module function settings are based on configuration according to user requirements

Step

Open Schedule For Recording

   1.  Loggin to Gen21 CMS
   2.  Open Module Schedule For Recording  ( Media Library --> Schedule For Recording )        

Searching

Searching is a facility for searching data that has been created, and the search is tailored to  the needs of the user based on existing criteria, for example Channel, Media Category, Record Date, Server Name, File Name, Program,  etc

                    

   

Add Schedule For Recording

1.  Press Right Button for open the popup menu

2.  Press edit in popup menu

3.  Press add in popup menu

4.  Enter the values in modifiable fields / Insert Data

5.  Press save from main men

 

Delete Schedule For Recording

1.   Press Right Button from program detail for open the popup menu

2.   Press mouse right button for open the popup menu

3.   Select the record in list

4.   Press Edit in Popup menu

5.   Press Delete in Popup menu 
6.   Press Save from Main menu 

                      Browse

                  Also, this menu has “Browse Tab” to display all (nama menu) that be existed in the system.User can find the specific (nama menu) by filter the name filter facility.

Steps on the browser screen

-  Click on browse tab

-  Find specific (nama menu) filter on the drop down filter list feature

-  Type on the text box filter of (nama menu) that want to search
