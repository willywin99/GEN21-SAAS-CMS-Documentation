Vendor

       Vendor

1.1.   Description

Creation and maintenance of detailed Vendor definition.

1.2.   Application Configuration

               Module function settings are based on configuration according to user requirements

1.3.   Steps

                Open Parameter Vendor

1.       Loggin to Gen21 CMS

2.       Open the module Vendor  ( Program Management --> Manage Episode Detail  --> Parameter --> Vendor )

      

                  Add Vendor

                  1.  Open Module Parameter Vendor
                  2.  Open the Vendor menu
                  3 .  Press edit in popup menu
                  4 .  Press add in popup menu
                  5 .  Code generated from system
                  6 .  Enter Name
                   7.  Enter Country
                  8 .  Enter City 
                  9 .  Enter Address
                10.  Enter Pos Code
                11.  Enter Phone Number
                12.  Enter Fax
                13.  Enter Email
                14.  Enter Contact Person
                15.  Press Save From Main Menu 

           

                  Delete Vendor

                  1. Open the Vendor menu
                  2 . Select the record in list
                  3 . Press Edit in Popup menu
                  4 . Press Delete in Popup menu 
                  5 . Press Save from Main menu

                If Record is allready used by transactions, Deletion can not possible
