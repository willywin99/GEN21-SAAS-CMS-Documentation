Currency

Currency

Descriptions

              Used for creating and maintenance of currency records

Application Configuration

               Module function settings are based on configuration according to user requirements

Steps

Open Parameter Currency

1.       Loggin to Gen21 CMS

2.       Open the module Currency  ( Program Management --> Parameter --> Currency )

           

            
               Add Currency

  1.  Open The Currency Window

  2. Press mouse right button for open the popup menu

  3.  Press edit popup menu

  4.  Press add in popup menu

  5.  Enter code of currency

  6.  Enter the name for currency

  7.  Enter non active date

  8.  Enter SAP Code

  9.  Selecet Default or not
 10 Press Save from main menu

                Delete Currency

1. Open the Currency window

2. Press mouse right button for open the popup menu

3. Select the record in list

4. Press Edit in Popup menu

5. Press Delete in Popup menu 

6. Press Save from Main menu

                   If Record is allready used by transactions, Deletion can not possible
