Country

Country

Descriptions

     List of countries that will be used for contract creation and where the contracts come from.

Application Configuration

               Module function settings are based on configuration according to user requirements.

Steps

Open Parameter Country

 1.  Loggin to Gen21 CMS
 2.  Open Module Country  ( Program Management --> Parameter  --> Country )

 

Add Country

1.  Open Module Country 
2.  Open the Country Window

3.  Press Right Button for open the popup menu

4.  Press edit in popup menu

5.  Press add in popup menu

6.  Enter the name of country

7.  Enter the PPN %

8.  Enter the PPH %

9.  Enter the SAP Code

10. Press save from main menu

.

Delete Country

1. Open the Country window

2. Press mouse right button for open the popup menu

3. Select the record in list

4. Press Edit in Popup menu

5. Press Delete in Popup menu 

6. Press Save from Main menu

                   If Record is allready used by transactions, Deletion can not possible
