Media Format

Used to creating of Media Format definition. This is the variety of available format, brand and duration of the recording media, eg. Betacam 60", CD 30", DV cam 32" etc.

 

Open the Media Format

1. Login to Gen21CMS.

2. Open the Media Format. (Media Library => Media Format) 

 

Steps Adding Media Format

1. Open the Media Format

2. Create new Media Format,

    a. Navigate to Form Tab and click Insert button.

    b. Define the Code, Name, and Digital File.

    c. Click Save button.

 

Steps Editing Media Format

1. Open the Media Format.

2. Edit specific Media Format,

    a. On Browse Tab, select the appropriate Media Format.

    b. Double click on selected Media Format or Navigate to Form Tab and click Edit button.

    c. Edit Name and Digital File.

    d. Click Save button.

 

Steps Deleting Media Format

1. Open the Media Format.

2. Delete Media Format,

    a. On Browse Tab, select the appropriate Media Format.

    b. Double click on selected Media Format or Navigate to Form Tab and click Edit button.

    c. Click delete button.

    d. Click OK to confirm the deletion.
