Blank Media Cek In Out

         Blank Media Cek In Out

Description

This feature serves to fill in data that will be transacted for both borrowing and returning media to receive programs, backups, media services etc. This module also functions for transactions media status be it writte off, recycle, Damage, Stock. All transactions in this module are also recorded in history.

Application Configuration

Module function settings are based on configuration according to user requirements

Step

Open Blank Media Cek In Out

1.  Loggin to Gen21 CMS

2.  Open Module Blank Media Cek In Out  ( Media Library --> Blank Media Cek In Out )  

Add Blank Media Cek In Out

1.  Press Right Button for open the popup menu

2.  Press edit in popup menu

3.  Press add in popup menu

4.  Select Field Mandatory

5.  Press save from main menu
                        

 

 

 

Browse

Displays the list of records matching the input search criteria. Simply use the arrow buttons or click on the desired single record on the list, Also, this menu has “Browse Tab” to display all (nama menu)
that be existed in the system. User can find the specific (nama menu) by filter the name on the filter facility.
Steps on the browser screen :
      -  Click on browse tab
      -  Find specific (nama menu) filter on the drop down filter list feature
      -  Type on the text box filter of (nama menu) that want to search

          Checking In only;

1.  Return Date: Actual return date of the Check In.
2.  Media Status: Status of the returned tape. Acceptable values are
         - Program       (Tape is now a 'Program' tape)
         - Archive        (Tape is now an 'Archive' tape)
         - Damaged      (Tape is 'Damaged', not available for checkouts anymore.         
         - Recycle        (Tape is already used for material recording)
         - Blank           (Tape is still a blank tape)
3.  Check-In Remarks: Free formatted additional short description of the Check-In
     transaction not recorded elsewhere
4.  Cancel Last By Tape# button>: Triggers the cancellation of the last transaction
     made filtered by Media number. Applicable to both Check-Out or Check-In
     transactions.
5.  Cancel Last By Borrower button>: Triggers the cancellation of the last transaction
     made filtered by Borrower name. Applicable to both Check-Out or Check-In  
     transactions.
6.  History Recycling button >: To view of the media's previous recycling
     information.
