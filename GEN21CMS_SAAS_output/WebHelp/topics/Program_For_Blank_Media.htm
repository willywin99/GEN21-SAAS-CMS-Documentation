Program For Blank Media

Used to creating the 'Blank Program' definition. These are self-defined names & titles of programs (typically scheduled on grids) which blank tapes will be used for.

Open the Program For Blank Media

1. Login to Gen21CMS.

2. Open the Primary Subjects (Menu-> Media Library -> Parameters -> Program For Blank Media)

 

Steps Adding Program For Blank Media

1. Open Program For Blank Media

2. Create new Program For Blank Media,

      a. Right click on mouse button on the header field and choose

          “add row” to create new Program For Blank Media

b. Define the Program Name, Title Name and Blank Budget

      c.  Right click on mouse button and choose “save”

 

Steps Editing Program For Blank Media

1. Open Program For Blank Media

2. Edit specific Program For Blank Media,

      a. Right click on mouse button and choose “edit mode”.

  b. Edit the Program Name, Title Name and Blank Budget

      c.  Right click on mouse button and choose “save”

 

Steps Deleting Program For Blank Media

1. Open Program For Blank Media

2. Delete Program For Blank Media,

      a. Right click on mouse button and choose “edit mode”

      b. Right click on mouse button and choose “delete”
