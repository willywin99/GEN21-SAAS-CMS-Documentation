Exchange Rate

Exchange Rate

Description

The currency value that will be used in making the contract is according to the period.

Application Configuration

               Module function settings are based on configuration according to user requirements

Steps

                Open Parameter Exchange Rate

1.       Loggin to Gen21 CMS

2.       Open Exchange Rate Menu ( Program Management -> Parameters -> Exchange Rate )  

 

                Add Exchange Rate

                 1.  Open the Exchange Rate menu. 
                 2.   Click Edit Button.
                 3.   Click Insert Button.
                 4.  Fill in the Form.
                 5.  Select the Currency Code.
                 6.  Enter the Period From.
                 7.  Enter the Period To.
                 8.  Enter the Exchange Rate.
                 9.  Click Save Button.

      

                Delete Exchange Rate

                1.   Open the Exchange Rate menu.
                2.   On Browse Section, select Exchange Rate which one will be deleted.
                3.   Navigate to Form Section or double click on selected Exchange Rate.
                4.   Click Edit Button.
                5.   Click Delete Button.
                6.   Click OK to confirm the deletion.

                If Record is allready used by transactions, Deletion can not possible
