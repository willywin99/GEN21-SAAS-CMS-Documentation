Exchange Rate

Exchange Rate

Description

The currency value that will be used in making the contract is according to the period.

Application Configuration

               Module function settings are based on configuration according to user requirements

Steps

                Open Parameter Exchange Rate

1.       Loggin to Gen21 CMS

2.       Open Exchange Rate Menu ( Menu-> Program Management -> Parameters -> Exchange Rate )  

                Add Exchange Rate

                 1.  Open the Exchange Rate window
                 2.  Press mouse right button for open the popup menu
                 3.  Press Edit in Popup menu
                 4.  Press Add in Popup menu
                 5.  Select the Currency Code
                 6.  Enter thePeriod From
                 7.  Enter the Period To
                 8.  Enter the Exchange Rate
                 9.  Press Save from Main menu

      

                Delete Exchange Rate

                1.   Open the Exchange Rate window
                2.   Press mouse right button for open the popup menu
                3.   Select the record in list
                4.   Press Edit in Popup menu
                5.   Press Delete in Popup menu 
                6.   Press Save from Main menu

                If Record is allready used by transactions, Deletion can not possible
