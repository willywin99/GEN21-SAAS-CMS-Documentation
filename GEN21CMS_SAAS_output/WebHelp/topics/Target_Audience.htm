Target Audience

Purpose   

            To add and delete episode's target audience information.

            Steps to proceed

1.      Click on the Program Management > Manage Episode Details > Episode Details > Target Audience menu item.  

2.      To add new Target Audience, at the Target Audience section and select Edit or Add, enter the values in modifiable fields then click the save button 

3.      To modify Target Audience, select the media from the browse list, at the Target Audience section select Edit, retype over any modifiable fields and click the save button .  

4.      To delete a Target Audience, select the media from the browse list, at the Target Audience section select Edit, at the Target Audience section and select delete.

Note: Deletions are not possible when the records in question are already used in relation to any other entities in the application.
