Generate Shelf

Used for creating the 'Shelf Number' definition. Often referred as 'Print Location' these are the actual physical shelf numbers where the media are to be placed in the library. The shelf # is available for use ONLY for the media it is created for (determined by its Media Type).

Open the Generate Shelf

1. Login to Gen21CMS.

2. Open the Primary Subjects (Menu-> Media Library -> Parameters -> Generate Shelf)

 

Steps Adding Generate Shelf

1. Open Generate Shelf

2. Create new Generate Shelf,

      a. Right click on mouse button on the header field and choose

          “add row” to create new Generate Shelf

b. Define the Building, Room, Almira, Rack No, Capacity From, Capacity To, Media Type and Location Desc

      c.  Right click on mouse button and choose “save”

 

Steps Deleting Generate Shelf

1. Open Generate Shelf

2. Delete Generate Shelf,

      a. Right click on mouse button and choose “edit mode”

      b. Right click on mouse button and choose “delete”

Generate Shelf window contains two main tab windows

Browse -> Display all Existing Shelf Record

Form -> Create new shelf or Edit existing shelf
