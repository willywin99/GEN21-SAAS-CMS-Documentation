Parameter

To record all the master parameters (static variables) required specifically by Media Library module of Gen21 Inventory. e.g. Media Format, Media Borrower, Create Shelf Number etc. These parameters are used throughout the module in dropdown (selection) list of related fields in all data entry/inquiry activities.

Parameters has following sub modules:

  • Attribute     
  • Archive Keywords   
  • Primary Subject 
  • Program Media Type
  • Blank Media Type
  • Media Brand
  • Purpose
  • Media Format
  • Departments
  • Media Borrower
  • Program for Blank Media 
  • Program for Archive 
  • Service Types
  • Feed
  • Server Schedule  
  • Generate Shelf #
  • Star / Crew Types
  • Stars / Crew Info
