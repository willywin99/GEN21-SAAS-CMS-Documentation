Receive Program Media

Receive Program Media

Receive Program is a module for material usage transactions, whether it is a file (digital) or a cassette, with details of storage shelf, cabinets, media type, type used, Barcode, Machine,until the making of the House Number, which will be used to fill in material, be it programs, promos, spots.

  Application Configuration

Module function settings are based on configuration according to user requirements

Step Open Receive Program Media

   1.  Login to Gen21 CMS
   2.  Open Module Receive Program Media  ( Media Library --> Receive Program Media  
               

Searching

Searching is a facility for searching data that has been created, and the search is tailored to the needs of the user based on existing criteria, for example by Shelf, House#, Blank  Media#, Program Name, Tittle Name, Borrower. Media Format, etc.  

Step Search

  1. check the all media checkbox to find all receive program media
                

 

Add Receive Program Media Header

     1.  Press Right Button for open the popup menu

     2.  Press edit in popup menu

     3.  Press add in popup menu

     4.  For Digital File Enter Duration, Media Format, Machine, and File Name

     5.  For Cassette enter Duration, Media Format, Media Type,

     6.  Check mark the auto selection list or details for filling in the house number

     7.  Press save from main menu

.

   Add Receive Program Media Detail

   1.   Press Right Button from program detail for open the popup menu
   2.   Press edit the popup menu
   3.   Add Multiple Episode

   4.   Select the record in filter ( PO, Package, Name etc )

   5.   Select Arr Date
   6.   Insert Data 
   6.   Press Save from Main menu

    Delete Receive Program Media

   1.   Press Right Button from program detail for open the popup menu

   2.   Press mouse right button for open the popup menu

   3.   Select the record in list

   4.   Press Edit in Popup menu

   5.   Press Delete in Popup menu 
   6.   Press Save from Main menu  

          

                        Browse

                   Also, this menu has “Browse Tab” to display all (nama menu) that be existed in the system.User can find the specific (nama menu) by filter the name on the filter facility.
                    Steps on the browser screen
                      -  Click on browse tab
                      -  Find specific (nama menu) filter on the drop down filter list feature
                       -  Type on the text box filter of (nama menu) that want to search
