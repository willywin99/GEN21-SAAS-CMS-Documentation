Cost Process Setting


 

Description

Used for creating and maintenance of Cost Process Setting. It is used by cost process Additional cost.

 

Application Configuration

 

          Module function settings are based on configuration according to user requirements. Also, menu will be enabled depends on version of application  

 

Steps

                            Open Parameter Cost Proses Setting

                     1.  Loggin to Gen21 CMS  
                     2.  Open Cost Process Setting Menu ( Program Management --> Parameter
                           --> Cost Process Setting )


 

                Add Cost Process Setting

                1. Open the Cost Process Setting menu. 
                2. Navigate to Form Tab. 
                3. Click Edit Button. 
                4. Click Insert Button.
                5. Fill in the Form.
                6. Click Save Button.
 

                Delete Cost Process Setting

                1.   Open the Cost Process Setting menu.
                2.   On Browse Tab, select the Cost Process Setting which one will be deleted.
                3.   Navigate to Form Tab or double click on selected Cost Process Setting.
                4.   Click Edit Button.
                5.   Click Delete Button.
                6.   Click Save Button.

                *If Record is allready used by transactions, Deletion can not possible
