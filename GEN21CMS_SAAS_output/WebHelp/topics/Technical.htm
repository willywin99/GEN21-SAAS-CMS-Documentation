Technical

To facilitate BOTH technical and content-related activities on all previously received media (mainly Programs) in the Media Library.

 

Note : Quality Control can be processed as long as it has gone through the process of receiving media

 

Open the Technical

 

1.    Login to Gen21CMS

2.    Open the Program Purchase Contract (Menu-> Program Management-> program Purchase Contract-> Program Info)

3.    In search Tab, Press Search Button

4.    In Browse tab, Select the PO and Open the Form Tab

5.    Press Button Technical
