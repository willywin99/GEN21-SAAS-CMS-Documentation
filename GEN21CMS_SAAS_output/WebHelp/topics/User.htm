User

Used to create and maintain all user information.

Basic Setup for User

Got Access Admin rights for EDP menu to GEN21 CMS.

Open the User

  1. Log in to GEN21 CMS.
  2. Open the User. (Master - User).

User menu contains 6 sections

  1. Browse  = > List of all user. 
  2. Form = > Form to Create User. < /FONT >
  3. User Group = > User group(s) assign to this user.< /FONT >
  4. User Channel = > Channel(s) assign to this user.
  5. Audit Trail Login => User session history.
  6. History => User Group History, Channel History, Region History.

Steps for Create User

  1. Open the GEN21 CMS .
  2. Open the User menu.
  3. Navigate to Form Tab.
  4. Click Insert button or CTRL + i.
  5. Fill the Form.
  6. After Email is filled, click generate password and keyword to get them.
  7. Select the Option for change the password on log on.
  8. Click Save button or CTRL + S.

Steps for adding the User Group

  1. Open the GEN21 CMS.
  2. Open the User menu.
  3. Select the user.
  4. Select User Group which one will be assign to selected user.
  5. Click Save button or CTRL + S.

Steps for adding channel access to user

  1. Open the GEN21CMS.
  2. Open the User menu.
  3. Select the user.
  4. Select Channel which one will be assign to selected user.
  5. Click Save button or CTRL + S.
