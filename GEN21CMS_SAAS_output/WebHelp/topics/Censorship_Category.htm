Censorship Category

Used to To define and maintain all censorship categories for the material. Category Code & Description are self-defined, but typical values associated with these are BO = Bimbingan Orang Tua, DW = Dewasa etc. 

Open the Censorship Category

  1. Login to Gen21CMS
  2. Open the Application Configuration (Menu-> Quality Control -> Content -> Censorship Category).

Steps Adding New Censorship Category

1. Open Censorship Category

2. Create new Censorship Category,

        a. Right click on mouse button on the header field and choose

                 “add row” to create new Censorship Category

        b. Define the Code and Category Description

        c. Right click on mouse button and choose “save”

 

Steps Editing Censorship Category

1. Open Censorship Category

2. Edit specific Censorship Category,

        a. Right click on mouse button and choose “edit mode”.

        b. Edit the Code and Category Description

        c. Right click on mouse button and choose “save”

 

Steps Deleting Censorship Category

1. Open Censorship Category

2. Delete Censorship Category,

        a. Right click on mouse button and choose “edit mode”

        b. Right click on mouse button and choose “delete”
