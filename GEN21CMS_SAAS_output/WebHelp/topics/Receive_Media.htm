Receive Media

Typically performed upon receipt of new materials, this facility creates initial physical 'Program' information in the inventory and then maintains it. 'Programs' are pre-recorded,ready-to-air material scheduled to run as 'shows' in the Slots. These Programs are purchased from either external sources (eg. Film Distributors) or from internal ones (eg. the station's own In-House production Dept), in either case formal Purchase Orders for them do exist and must have already been issued before they can be 'received' at this point. Depending on configuration, the Tape Library may also receive other 'ready' material types eg. 'Commercials', 'Promos' etc.

 

Open the Receive Media

1.    Login to Gen21CMS

2.    Open the Program Purchase Contract (Menu-> Program Management-> program Purchase Contract-> Program Info)

3.    In search Tab, Press Search Button

4.    In Browse tab, Select the PO and Open the Form Tab

5.    Press Button Receive Media
