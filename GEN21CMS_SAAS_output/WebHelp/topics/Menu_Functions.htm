3. Find/ Filter & Sort Data

In Gen21, Searching, Finding and Filtering of data is very easy. Each form shows the following section:

No

Description

1

Column List - as soon as any column in selected to search or filter, the data gets sorted on that column

2

User can write - and the context sensitive search for the column selected in section 1 gets activated. Sorting is not case sensitive

3

User can write and if the 2 and 3 words exactly meet any criteria in column selected in section 1, the data gets filtered. This filtering is case sensitive
