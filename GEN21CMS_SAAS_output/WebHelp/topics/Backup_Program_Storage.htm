Backup Program Storage

             To store the program after airing, with some additional fields like Blank Media #and Media Type, and processes like Print To Media. It will write the program contents directly to the blank tape and blank tape stock  will be reduced for that tape type automatically. User also may add/edit the additional programs/episodes information.

Note: Before doing the backup process, make sure the storage program has done the process of receiving blank and blank media check in / out.

 Steps to proceed

     1.    Click on the Media Library > Backup Program Storage  menu item. The function startswith a 'Search' screen to locate the media to be worked on.Enter any search criteria(s) in  available fields and click 'Search'  button to commence the search. Alternately, click 'Clear' button to erase any criterias already-entered in all fields.

 

 

 

 

      2. The main Backup Program Storage ('Form') screen is displayed, by default showing the first record of the search

 

 

 

 

At this point one may also view all records found by the search operation by clicking the 'Browse' tab, or click 'Search' tab to perform  another (different search:      

3.    To add new Backup Media, right click at the header of the Form screen and select Edit, right click at the header again and select Add,

    enter the values in modifiable fields then click the save button .

 

4.    To modify Backup Media, select the media from the browse list , right click at the header of Form tab and select Edit,

    retype over any modifiable fields and the save button . 

 

            5 .  To delete a Backup Media, select the media from the browse list, right click at the header and select Edit, right click again at the header and delete. 
 

 

  Note: Deletions are not possible when the records in question are already used in relation to any other entities in the application.
