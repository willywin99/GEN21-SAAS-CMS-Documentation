Payment Schedule

To create the schedule of payment that has to be done for particular PO#. For one payment schedule, there can be more than one payment term. After some periode time, the schedule will be approved and there will be no more changes can be done on that schedule. If should any changes need to be done, then the schedule need to be Revised. The approval or revise schedule can be done only by authorized person only.

 

Open the Payment Schedule

1. Follow the steps outlined in Program Purchase Contract to get to the main Program Purchase Contract (Form) screen & c lick onthe Payment Schedule menu item.

2.To add new Payment Schedule, right click at the header and select Edit, right click at the header again and select Add, enter the values in modifiable fields then click the save button.

3. To modify Payment Schedule, select the media from the browse list, right click at the header of Form tab and select Edit, retype over any modifiable fields and click the save button . 

4. To delete Payment Schedule, select the media from the browse list, right click at the header and select Edit, right click again at the header and select delete.

                                                                                                                

Note: Deletions are not possible when the records in question are already used in relation to any other entities in the application.
