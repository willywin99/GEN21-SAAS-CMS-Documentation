Media Borrower

Used to creating the Media Borrower definition. This is the listing of (registered) users and their functional departments that are authorized to borrow/return tapes to the library.

Open the Media Borrower

1. Login to Gen21CMS.

2. Open the Primary Subjects (Menu-> Media Library -> Parameters -> Media Borrower)

 

Media Borrower window contains three main tab windows

1.    Search tab -> Used for search the existing Records

2.    Browse tab -> Used to see the All existing records or listing based on search criteria in search tab

3.    Form tab -> Used to create new borrower details or edit the existing borrower details

 

Form Tab

Browse Tab

Form tab divided into three sections

1. Header Section -> For Borrower details

2. Channel Section -> Define the channel (Used for multi channel)

3. Validation Section -> Define the type of media, no of credit days and

    maximum Borrowing

 

Steps Adding Media Borrower

1. Open Media Borrower

2. Create new Media Borrower,

      a. Right click on mouse button on the header field and choose

          “add row” to create new Media Borrower

b. Define the Name, NIK, Department, Division, Address, Phone, Non Active Date, Blank List, Resign, Email, Allow Copy Digital File and Head

      c.  Right click on mouse button and choose “save”

 

Steps Editing Media Borrower

1. Open Media Borrower

2. Edit specific Media Borrower,

a. Right click on mouse button and choose “edit mode”.

  b. Edit the Name, NIK, Address, Phone, Non Active Date, Blank List, Resign, Email, Allow Copy Digital File and Head

      c.  Right click on mouse button and choose “save”

 

Steps Deleting Media Borrower

1. Open Media Borrower

2. Delete Media Borrower,

      a. Right click on mouse button and choose “edit mode”

      b. Right click on mouse button and choose “delete”
