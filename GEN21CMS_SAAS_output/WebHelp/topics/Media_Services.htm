Media Services

          

        Media Services

         Media Services 

Description

To store all about services information in Media Library. Services Name will come from master of services and Media Library can create different services nameso further can use them in the services interface while providing the service.

Application Configuration

Module function settings are based on configuration according to user requirements

Step

Open Media Services

1.  Loggin to Gen21 CMS
2.  Open Module Media Services  ( Media Library --> Media Services ) 
 

Searching

Searching is a facility for searching data that has been created, and the search is tailored to  the needs of the user based on existing criteria, for example Borrower, Borrower Head, Departement, Service#, Program Name, Blank Media#.  
                                         

 

Add Media Services

1.  Press Right Button for open the popup menu

2.  Press edit in popup menu

3.  Press add in popup menu

4.  Enter the values in modifiable fields / Insert Data

5.  Press save from main menu

 

Delete Media Services

1.   Press Right Button from program detail for open the popup menu

2.   Press mouse right button for open the popup menu

3.   Select the record in list

4.   Press Edit in Popup menu

5.   Press Delete in Popup menu 
6.   Press Save from Main menu 

 

 

Browse

Also, this menu has “Browse Tab” to display all (nama menu) that be existed in  the system.User can find the specific (nama menu) by filter the name on the filter facility.

Steps on the browser screen

-  Click on browse tab

-  Find specific (name menu) filter on the drop down filter list feature

-  Type on the text box filter of (nama menu) that want to sear
