Media Library

The Media Library Module of Gen21 Inventory specifically deals with most of the common tasks of the Media Library department, which mainly consist of Receiving and Circulation of various

material types (Blank, Program, Archive- can be extended to accommodate more types like Promo & Commercial).  To support day-to-day needs of the Media Library department,

various standard reports associated with these tasks are available as well as a number of standard, flexible built-in formatting features like Sorting, various Format Saving and Filtering.

As a standard feature common to every Gen21 suite of products module, the Media Library module utilizes extensive Parameterization features incorporated into its

Master definition facilities that not only minimizes key strokes and human errors but also ensures optimum data integrity across database tables and user interface.
