Transcritpion / QC Status

Used to create the status of Media. Possible values are Reject, Revise, Pass or Pending. The statuses of the media are user defined, which means users are able to create more statuses as the need arises.

Open the Transcription/QC Status

  1. Login to Gen21CMS.
  2. Open the Application Configuration (Menu-> Quality Control -> Content -> Transcription/QC Status).
