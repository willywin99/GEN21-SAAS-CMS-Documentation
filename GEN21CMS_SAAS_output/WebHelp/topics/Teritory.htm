Teritory

Teritory

Description

 

Creation and maintenance of Territory definition. Territories are defined as the broadcaster's separate operational coverage based on distinct predefined entities (e.g. Geography, Companies, Market).

note : menu will appear depends on version of application 

Application Configuration

 

       Module function settings are based on configuration according to user requirements. Also, menu will be depends on version of application 

Steps

                            Open Parameter Teritory

1.        Loggin to Gen21 CMS

2.       Open Teritory Menu ( Program Management -> Parameters -> Teritory )

               

Add Teritory

                1. Open the Teritory menu.
                2. Navigate to Form Tab.
                3. Click Edit Button.  
                4. Click Insert Button.
                5. Fill in the Form.
                6. Click Save Button.
 

               

Delete Source

                1.   Open the Teritory menu.
                2.   On Browse Tab, select the Source which one will be deleted.
                3.   Navigate to Form Tab or double click on selected Source.
                4.   Click Edit Button.
                5.   Click Delete Button.
                6.   Click Save Button.

                If Record is allready used by transactions, Deletion can not possible
