Dubbing Process

Dubbing Process 

 

1.   Descriptions

 

Facility to define the details of the dubbing process, and the name is filled according to dubbing needs, eg translation, dialog writing, dubbing localization etc.

2.   Application Configuration

 

Module function settings are based on configuration according to user authorization on the master > process authorization

1.      P015 – Authorization for Dubbing Epi

 

3. Steps

 

Open Dubbing Process

1.    Login to Gen21 CMS

2.    Open the module Dubbing  ( Program Management --> Manage Episode --> Parameter --> Dubbing Process)                  

Add Dubbing Process

1.   Open the Dubbing Window

3.   Press add button in popup menu or right click and select Add Row menu

4.   Input Proses Name

5.   Press save button from main menu or (CTRL + S)

6.   The code will be generated from system

 

Edit Dubbing Process

1.   Open the Dubbing window

2.   Select the item that being to edit in the list

3.   Press Edit button in popup menu or right click and select Edit Mode menu

4.   Correct the process name

5.   Press save button from main menu or (CTRL + S)

 

Delete Dubbing Process

1.    Open the Dubbing window

2.    Select the item that being to edit in the list

3.    Press Edit button in popup menu or right click and select Edit Mode

4.    Press Delete button in popup menu or right click and select Delete Row menu

5.    Press save button from main menu or (CTRL + S)

Please note, If Record is already used on the transaction, The deletion can not possible
