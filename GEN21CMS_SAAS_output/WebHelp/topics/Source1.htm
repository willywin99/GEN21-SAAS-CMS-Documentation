Source

Source

 

Description

The source / origin of the purchase is either local or imported.

 

Application Configuration

 

          Module function settings are based on configuration according to user requirements.

 

Steps

Open Parameter Source

1.        Loggin to Gen21 CMS

2.         Open Source Menu ( Program Management -> Parameters -> Source )

Add Source

1. Open the Source menu. 
2. Navigate to Form Tab.
3. Click Edit Button.
4. Click Insert Button.
5. Fill in the Form.
6. Click Save Button.

Delete Source

1.   Open the Source menu.
2.   On Browse Tab, select the Source which one will be deleted.
3.   Navigate to Form Tab or double click on selected Source.
4.   Click Edit Button.
5.   Click Delete Button.   
6.   Click OK to confirm the deletion.

*If Record is already used by transactions, Deletion can not possible.
