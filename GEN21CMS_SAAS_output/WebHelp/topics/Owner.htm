Owner

Owner

Description

Used for creating and maintenance of Owner. It is used by Assign Owner

note : menu will appear depends on version of application.  



Application Configuration

Module function settings are based on configuration according to user requirements

Steps

                            Open Parameter Owner

                     1.  Loggin to Gen21 CMS  
                     2.  Open Owner Menu( Program Management --> Parameter  --> Owner )

 

                Add Owner

                1. Open the Owner menu.
                2. Navigate to Form Tab. 
                3. Click Edit Button. 
                4. Click Insert Button. 
                 5. Fill in the Form.
                6. Click Save Button.


                Delete Owner

                1.   Open the Owner window
                2.   Press mouse right button for open the popup menu
                3.   Select the record in list
                4.   Press Edit in Popup menu
                5.   Press Delete in Popup menu 
                6.   Press Save from Main menu

                *If Record is allready used by transactions, Deletion can not possible
