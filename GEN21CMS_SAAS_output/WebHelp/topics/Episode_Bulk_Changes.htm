Episode Bulk Changes

To change mutiple items in an episode.

 

Steps to proceed

1.  Click on the Program Management >  Program Purchase Contract > Program Info > Episode Bulk Changes menu item.

2.  To add Episode Bulk Changes, enter the values in modifiable fields then click the OK button.
