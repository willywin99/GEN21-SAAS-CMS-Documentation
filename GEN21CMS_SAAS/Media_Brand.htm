Used to Creating the 'Media Brand' definition. These are self-defined codes & names of brand which blank Medias will be used for.

Open the Media Brand.

1. Login to Gen21CMS.

2. Open the Media Brand (Media Library => Parameters => Media Brand. 

 

Steps Adding Media Brand

1. Open the Media Brand.

2. Create new Media Brand,

    a. Navigate to Form Tab and click Insert button to create new Media Brand.

    b. Define the Brand Code and Brand Name.

    c. Click Save button.

 

Steps Editing Media Brand

1. Open the Media Brand.

2. Edit specific Media Brand,

    a. On Browse Tab, select the appropriate Media Brand.

    b. Double click on selected Media Brand or Navigate to Form Tab and click Edit button.

    c. Edit the Brand Name.

    d. Click Save button.

 

Steps Deleting Media Brand

1. Open the Media Brand

2. Delete Media Brand,

    a. On Browse Tab, select the appropriate Media Brand.

    b. Double click on selected Media Brand or Navigate to Form Tab and click Edit button.

    c. Click Delete button.

    d. Click OK to confirm the deletion.
