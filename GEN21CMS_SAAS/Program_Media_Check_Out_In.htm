Program Media Check-In/Out

Purpose

Working in synchronization with Receive Program Media , this function manages the circulation (borrow & return) of program material by appropriate users in the organization.
 
Note: Material must be 'received' before any Check-In/Out action can be initiated.
Steps to proceed  

  1. Click on the Media Library > Program Media Check-In/Out  menu item. The starting Program Check-In/Out screen is displayed:

    As before, the function starts with a 'Search' screen to locate the media to be worked on. Enter any search criteria(s) in available fields in the left subwindow  and click 'Search' button to commence the search. Alternately, click 'Clear' button to erase any criterias already-entered in all fields.
  2. The output of the search is a list of all records found by the search, shown in 2 possible formats depending on the selected Tab at the bottom of the left sub window ('Grid'). The default is on the 'Grid' form (see clicked tab under each display below). At this point one may also click 'Search' tab to perform another (different) search.
  3. At the Grid form view, select 1 or more media to be worked on by clicking them individually in the leftmost 'X' column (or select/deselect ALL by clicking the 'X' column heading).Please observe items currently In or Out (colour indicated) while making your selections. The screen should now appear with the (set of) media already selected like this:  
  4. Perform specific activities on the selected records by clicking any of the available functions listed as the horizontal yellow bars in the bottom of the right subwindow. Please refer to the following dedicated subsections for details on each chosen activity:
    1. Cancel Last Transaction
    2. Check Out
    3. Check In
    4. Media Usages
    5. Media Properties
    6. Historical Activities
