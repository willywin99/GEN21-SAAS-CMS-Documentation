Multiple Write Off

Description

 This feature serves to fill in data that will be transacted for both borrowing and
 returning media to receive programs, backups, media services etc. This module
 also functions for transactions media status be it writte off, recycle, Damage, Stock.
 all transactions in this module are also recorded in history.

Application Configuration

Module function settings are based on configuration according to user requirements

Step

Open Multiple Write Off

1.  Loggin to Gen21 CMS

2.  Open Module Multiple Write Off  ( Media Library --> Multiple Write Off  )

Add Multiple Write Off

1.  Press Right Button for open the popup menu

2.  Press edit in popup menu

3.  Press add in popup menu

4.  Select Field Mandatory

5.  Press save from main menu
                                             

 

                       Searching

                         Searching is a facility for searching data that has been created, and the search is tailored to  the needs of the user based on existing criteria, for example by Shelf, House#, Blank 
                        Media#, Program Name, Tittle Name, Borrower. Media Format, etc.

                       

                      Browse

                  Displays the list of records matching the input search criteria. Simply use the arrow buttons or click on the desired single record on the list, Also, this menu has “Browse Tab” to display all (nama menu) that be existed in the system. Use can find the specific (nama menu ) by filter the name on the filter facility.
                  Steps on the browser screen :

-  Click on browse tab

-  Find specific (nama menu) filter on the drop down filter list feature

-  Type on the text box filter of (nama menu) that want to search
