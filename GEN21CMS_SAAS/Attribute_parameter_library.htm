Used to Creating the Attribute definition. this to add the attribute in backup program.

Open the Attribute

1. Login to Gen21CMS.

2. Open the Primary Subjects (Menu-> Media Library -> Parameters -> Attribute)

 

Steps Adding New Attribute

1. Open Attribute

2. Create new Attribute,

              a. Right click on mouse button on the header field and choose

                 “add row” to create new Attribute

              b. Define the Attribute Name

              c.  Right click on mouse button and choose “save”

 

Steps Editing Attribute

1. Open Attribute

2. Edit specific Attribute,

              a. Right click on mouse button and choose “edit mode”.

              b. Edit the Attribute Name

              c.  Right click on mouse button and choose “save”

 

Steps Deleting Attribute

1. Open Attribute

2. Delete Attribute,

              a. Right click on mouse button and choose “edit mode”

              b. Right click on mouse button and choose “delete”
