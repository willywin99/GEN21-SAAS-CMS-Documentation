Major Code

 

Description

Used for creating and maintenance of Major Code. It is used by PO.

Application Configuration

               Module function settings are based on configuration according to user requirements

Steps

                Open Parameter Major Code

1.       Loggin to Gen21 CMS

2.       Open Major Code Menu ( Program Management -> Parameters -> Major Code )

                Add Major Code

                Major code setting according to station agreement, data is created directly from the database.

                Delete Major Code

                Delete Major code must be as agreed, because the data is immediately deleted from the
                database
