Open Change Password Menu

  1. Login to GEN21 CMS.
  2. Open Change Password page. (Hover on icon before Master menu and click Change Password)

  3. Enter your current password, new password and repeat password to confirm this new password.
  4. Click Change Password button to Submit.

 

Validations for Password Change

  1. Valid passwords contain a minimum of 6 (six) to a maximum of 10 (ten) alphanumeric characters with no spaces. Special characters are allowed.
  2. For security reason characters typed are appearing as series of '*' and length adjusted.
  3. This facility only work on passwords. User's Login Ids (User Ids) are not changeable once its entered in the system. (See next section, User Master)
  4. When an  old password is forgotten, System Administrators can override this setting and manually create a new password for the user. (See next section, User Master)
  5. For security reason, User will be blocked from accessing the system upon 3 (three) unsuccessful login.
  6. System Administrators can rectify the situation by individually 'unblocking' the user.
