Linear Rights Bulk Changes is to determine the territory and right of each episode based on license.

 

Open the Linear Rights Bulk Changes

1.    Login to Gen21CMS

2.    Open the Program Purchase Contract (Menu-> Program Management-> program Purchase Contract-> Program Info)

3.    In search Tab, Press Search Button

4.    In Browse tab, Select the PO and Open the Form Tab

5.    Press Button Linear Rights Bulk Changes
