Manage Media Relocation

          Manage Media Relocation

1.1          Description

This module functions for the transfer of program material / archive transactions for both return media, relocate and recycle.

1.2          Application Configuration

Module function settings are based on configuration according to user requirements

  Step

   Open Manage Media Relocation

   1.  Loggin to Gen21 CMS
   2.  Open Module Manage Media Relocation  ( Media Library --> Manage Media Relocation  
                                                                                                                                                              

    Add Manage Media Relocation

     1.  Select the data that will appear, whether a program or archive

     2.  Select Filter PO#

     2.  Select PO# in Pop Up Window

     3.  Press Button Ok

     4.  Select it and move it to a new shelf

     5.  Press save from main menu

.

 

                      Function Button

         -       Relocate    :  Move material from old shelf to available shelf.
    -  Return       :  Returns borrowed or outgoing material
    -   Recycle     :  Recycle filled material for reuse
