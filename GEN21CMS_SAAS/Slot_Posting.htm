Used to post all the programs aired for the selected date, so that the amortization is calculated. Using this module revenue is also posted for each run based on the revenue booked using BMS.

Note: Scheduling can be done by recording the process of the purchase contract program, receive media, quality control and allocate house number

Open the Slot Posting

  1. Login to Gen21CMS.
  2. Open the Slot Posting (Menu-> Scheduling-> Slot Posting).

 

 

 

Slot Posting Window is divided into 2 Section.

  1. Left Section shows all the slot defined in scheduling module and episodes allocated to each slots.
  2. Right Section shows the data fromas run log imported into Gen21.
  3. As run log is the actual airing data (programs) received from Master Control.

 

Steps for Slot Posting

  1. Open the Slot posting window
  2. Select the Channel name
  3. Select the Airing Date
  4. Press Retrieve Button
