Used to Creating the 'Primary Subjects' definition. These are one word subject descriptions that label the stored media to describe its contents and therefore facilitate quicker access to the right media.Each media can only be labeled with 1 (one) Primary Subject.

Open the Primary Subjects

1. Login to Gen21CMS.

2. Open the Primary Subjects (Media Library => Parameters => Primary Subjects)

 

 

Steps Adding Primary Subjects

1. Open Primary Subjects

2. Create new Primary Subjects,

    a. Navigate to Form Tab and click Insert button to create new Primary Subjects.

    b. Define the Code and Primary Subjects

    c. Click Save button.

 

 

Steps Editing Primary Subjects

1. Open Primary Subjects

2. Edit specific Primary Subjects,

    a. On Browse Tab, select the appropriate Primary Subjects, then navigate to Form Tab and click Edit button.

    b. Edit the Primary Subjects.

    c. Click Save button. 

 

 

Steps Deleting Primary Subjects

1. Open Primary Subjects

2. Delete Primary Subjects,

    a. On Browse Tab, select the appropriate Primary Subjects.

    b. Navigate to Form Tab and click Delete button.

    b. Click OK to confirm the deletion.
