Receive Stock Shoot

         Receive Stock Shoot 

Description

To create and maintain all pre-recorded materials classified as 'Stock Shoot' (ie. those obtained without a Purchase Order, typically self-created by the field operation crew). The function is typically performed upon receipt of
any recorded material produced from blank tapes earlier issued by the library

Application Configuration

Module function settings are based on configuration according to user requirements

  Step

   Open Receive Stock Shoot

   1.  Loggin to Gen21 CMS
   2.  Open Module Receive Stock Shoot  ( Media Library --> Receive Stock Shoot )
 

            
 

                  Searching

                   Searching is a facility for searching data that has been created, and the search is tailored to  the needs of the user based on existing criteria, for example Keywords, Primary subject,
                  Program, Duration or house#, Shelf, Blank Media#, etc

             

 

Add Receive Stock Shoot Header

1. Press Right Button for open the popup menu Header

2. Press edit in popup menu

3. Press add in popup menu

4. Enter the values in modifiable fields/Insert Data

5. Press save from main menu

 

Add Receive Stock Shoot Detail

1. Press Right Button for open detail

2. Press edit in popup menu

3. Press add in popup menu

4. Enter the values in modifiable fields/Insert data

5. Press save from main menu



Delete Receive Stock Shoot

1.   Press Right Button from program detail for open the popup menu

2.   Press mouse right button for open the popup menu

3.   Select the record in list

4.   Press Edit in Popup menu

5.   Press Delete in Popup menu  

6.   Press Save from Main menu
