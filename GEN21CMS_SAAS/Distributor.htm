Distributor

Description

The material provider recorded in the purchase / contract

Application Configuration

               Module function settings are based on configuration according to user requirements

Steps

                Open Parameter Distributor

1.       Login to Gen21 CMS

2.       Open Distributor Menu ( Program Management => Parameters = > Distributor )

         

                 Add Distributor

1.       Open the Distributor menu.

2.       Navigate to Form Tab.

3.       Click Insert Button.

4.       Enter The Name.

5.       Select the Country.

6.       Enter the address1 and addres2 and address3.

7.       Enter the Post Code.

8.       Enter the COD.

9.       Enter Phone number 1 & 2.

10.   Enter the Contact Person.

11.   Enter Email 1 & 2.

12.   Press Save button to add the data.

                Delete Distributor

                1.   Open the Distributor menu.
                2.   On Browse Tab, select the distributor which one will be deleted.
                3.   Navigate to Form Tab or double click on selected Distributor.
                4.   Press Edit Button.
                5.   Press Delete Button.
                6.   Click OK to confirm the deletion.



                If Record is allready used by transactions, Deletion can not possible
