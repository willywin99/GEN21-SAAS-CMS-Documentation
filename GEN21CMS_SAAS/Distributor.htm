Distributor

Description

The material provider recorded in the purchase / contract

Application Configuration

               Module function settings are based on configuration according to user requirements

Steps

                Open Parameter Distributor

1.       Loggin to Gen21 CMS

2.       Open Distributor Menu ( Menu-> Program Management -> Parameters -> Distributor )

         

                 Add Distributor

1.       Open the Distributor window

2.       Press mouse right button for open the popup menu

3.       Press Edit in Popup menu

4.       Press Add in Popup menu

5.       Enter The Name

6.       Select the Country Name

7.       Enter the address1 and addres2

8.       Enter the Pos kode

9.       Enter Non active Date

10.   Enter the COD

11.   Enter The Phone number

12.   Enter the Fax number

13.   Enter the contact person

14.   Enter Email

15.   Press Save from Main menu

                Delete Distributor

                1.   Open the Distributor window
                2.   Press mouse right button for open the popup menu
                3.   Select the record in list
                4.   Press Edit in Popup menu
                5.   Press Delete in Popup menu 
                6.   Press Save from Main menu

                If Record is allready used by transactions, Deletion can not possible
