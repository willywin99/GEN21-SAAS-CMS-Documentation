Used for maintaining Report Authorization(s) for each module/sub-module in the System. Authorization within a module (Report Authorization). This is used to give report access to user.

Basic Setup for Report Authorization

Got access Admin rights for EDP menu to GEN21CMS.

Open the Report Authorization

  1. Login to GEN21 CMS.
  2. Open Report Authorization menu (Master => Report Authorization).

 

 

Grant/Revoke Access to Report(s)

  1. Open Report Authorization menu.
  2. Open the Browse tab and select user.
  3. Open Form tab or double click on selected user.
  4. Drag and drop report from the list of User Authorization.
  5. To grant access, check the appropriated check box for each report on the list.
  6. To revoke access, uncheck the appropriated check box for each report on the list.
  7. Click save from the main menu.
