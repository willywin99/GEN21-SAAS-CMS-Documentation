Used to Apply Budget on the parameters that have been made in Manage Program Budgets

           Open The Program Approval Month End Closing

  1. Login to Gen21CMS.
  2. Open the Application Configuration (Menu-> Account -> Approval Month End Closing).

     Steps For Add Approval Month End Closing   

  1. Open the window Approval Month End Closin 
  2. Enter P.O.#  
  3. Enter Program Name
  4. Enter EpisodeName 
  5. Press Retrieve from Approval Month End Closing
  6. Check to select
  7. Press to Approval User 1 or Approval User 2
  8. Press to Process
