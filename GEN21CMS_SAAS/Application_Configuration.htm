Basic Setup for Application Configuration

Got Access Admin rights for EDP Menu to Gen21CMS 

Open the Application Configuration window

1.      Login to Gen21CMS.

2.      Open the Application Configuration (Menu-> Master -> Application Configuration).

Application Configuration window contains 3 sections Application configuration

1.      List of Application configuration based on groups.List of configurations 

2.      Display List of configuration based on selected group.

3.      Detail Configuration-> Details of Selected Configuration in List of Configurations.

Edit the Configuration Values 

1.      Press Edit from Top menu bar or Press Edit from Popup Menu or Press F4.

2.      Select the Config Code for Edit.

3.      Edit the Description or Edit Parameter value

4.      Press Save from Top Menu or Press Save from Popup menu or Press CTRL+S.
