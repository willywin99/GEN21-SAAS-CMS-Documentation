Schedule Type

Description

Used for creating and maintenance of Schedule Type. It is used by Payment Schedule.

Application Configuration

                Module function settings are based on configuration according to user requirements

Steps

                            Open Parameter Schedule Type

1.        Loggin to Gen21 CMS

2.         Open Schedule Type Menu ( Program Management -> Parameters ->     Schedule Type )

                Add Schedule Type

                1. Open the Schedule Type menu.
                2. Navigate to Form Tab.
                3. Click Edit Button.
                4. Click Insert Button.   
                5. Fill in the Form.
                6. Click Save Button.


 

                Delete Schedule Type

                1.   Open the Schedule Type menu.
                2.   On Browse Tab, select Schedule Type which one will be deleted.
                3.   Navigate to Form Tab or double click on selected Schedule Type.
                4.   Click Edit Button.
                5.   Click Delete Button.
                6.   Click OK to confirm the deletion.

                If Record is allready used by transactions, Deletion can not possible
