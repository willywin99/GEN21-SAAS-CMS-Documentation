Used to Creating the Program For Archive definition. this to add the Program Name in Receive Stock Shoot.

Open the Program For Archive

1. Login to Gen21CMS.

2. Open the Primary Subjects (Menu-> Media Library -> Parameters -> Program For Archive)

 

Steps Adding Program For Archive

1. Open Program For Archive

2. Create new Program For Archive,

      a. Right click on mouse button on the header field and choose

          “add row” to create new Program For Archive

b. Define the Program Name

      c.  Right click on mouse button and choose “save”

 

Steps Editing Program For Archive

1. Open Program For Archive

2. Edit specific Program For Archive,

      a. Right click on mouse button and choose “edit mode”.

  b. Edit the Program Name

      c.  Right click on mouse button and choose “save”

 

Steps Deleting Program For Archive

1. Open Program For Archive

2. Delete Program For Archive,

      a. Right click on mouse button and choose “edit mode"

      b. Right click on mouse button and choose “delete”
