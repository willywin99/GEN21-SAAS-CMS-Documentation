Steps for Login Gen21CMS

1.       Open the Application Gen21CMS

2.       Enter the user Name

3.       Enter the password

4.       Enter the Channel

5.       Press login Button 

 

Validations For Login

1.       User name should not be empty.

2.       Password should not be empty.

3.       Channal name should not be empty.

4.       User name should not be in Lock State.

5.       License for Channel should be valid.

6.       User should have Access right.
