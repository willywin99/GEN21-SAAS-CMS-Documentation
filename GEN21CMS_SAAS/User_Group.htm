Used to provides facility to create different User Groups depending on the modules.

Basic Setup for User Group

Got Access Admin rights for EDP menu to GEN21 CMS.

Open the User Group

  1. Log in to GEN21 CMS.
  2. Open the User Group (Master > User Group).

User group window contains 3 sections

  1. User group list = > List of all user. < /FONT >
  2. User group detail = > Display the details of selected user. < /FONT >
  3. User access function list = > Display the user group details of selected user. < /FONT >

Steps for create User Group

  1. Open the User Group menu.
  2. Navigate to Form Tab.
  3. Fill the Form.
  4. Select user feature.
  5. Click Save button or CTRL + S.

Steps for Modify the function list

  1. Open the User Group window.
  2. Select the User Group code from User Group list.
  3. Right click on User Group function detail section for open the pop-up window.
  4. Select the function from drop-down list box on function column.
  5. Click Save from the main menu.
