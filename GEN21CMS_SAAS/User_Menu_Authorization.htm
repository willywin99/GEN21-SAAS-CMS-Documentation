Used for maintaining User Menu Authorization(s) for each Module/Sub-Module in the System.

  1. User menu authorization(s) has 4 level of authorization module level. This is used to give access of the module to a user.
  2. Authorization within a module (Menu Authorization). This is used to give menu access to user.

Basic Setup for User Menu Authorization

Got access Admin rights for EDP menu to GEN21 CMS.

Open the User Menu Authorization

  1. Login to GEN21 CMS.
  2. Open User Menu Authorization. (Master => User Menu Authorization)

 

Grant/Revoke Access to Menu(s)

  1. Open User Menu Authorization.
  2. Open the Browse tab and select user.
  3. Open Form tab or double click on selected user.
  4. Drag and drop menu from the list of User Menu Authorization.
  5. To grant access, check the appropriated check box for each menu on the list.
  6. To revoke acces, uncheck the appropriated check box for each menu on the list.
  7. Click save from the main menu.
