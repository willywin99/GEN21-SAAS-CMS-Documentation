Used for create and maintain currency records.

Open Currency Menu

  1. Login to GEN21 CMS.
  2. Open Currency Menu. (Program Management => Parameters => Currency)

Steps for Add Currency

  1. Open Currency Menu.
  2. Navigate to Form Tab, click Insert Button or CTRL + I.
  3. Entry Currency Code.
  4. Entry the name of currenncy code.
  5. Entry Non Active Date.
  6. Entry SAP Code.
  7. Check Default check box if this currency is default.
  8. Click Save Button or CTRL + S.

Steps for Delete Currency

  1. Open Currency Menu.
  2. On Browse Tab, select the currency which one will be deleted.
  3. Navigate to Form Tab or double click on selected currency.
  4. Click Delete Button.
  5. Click OK to confirm the deletion.

*If Record is already used in transaction(s), deletion is not possible.
