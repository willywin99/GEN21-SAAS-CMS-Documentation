Program Sub Type

Description

Used for creating and maintenance of Program Sub Type records. These are the
'subtypes' of (each 'Program Category' defined earlier) the purchased material falls
 under (eg. Comedy, Horror). Normally used to further define the nature of the
'content' of the material.

Application Configuration

                 Module function settings are based on configuration according to user requirements

Steps

                            Open Parameter Program Sub Type

1.        Loggin to Gen21 CMS

2.       Open Program Sub Type Menu ( Program Management -> Parameters -> Program Sub Type )

                Add Program Sub Type

                1. Open the Program Sub Type menu.
                2. Navigate to Form Tab.
                3. Click Edit button.
                4. Click Insert button.
                5. Fill in the Form.
                6. Click Save button.


 

                Delete Program Sub Type

                1.   Open the Program Sub Type menu.
                2.   On Browse Tab, select Program Sub Type which one will be deleted.
                3 .  Click Edit Button.
                4 .  Click Delete Button.
                5 .   Click OK to confirm the deletion.

                If Record is allready used by transactions, Deletion can not possible
