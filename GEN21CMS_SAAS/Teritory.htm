Teritory

Description

             Creation and maintenance of Territory definition. Territories are defined as the
             broadcaster's separate operational coverage based on distinct predefined entities
             (e.g. Geography, Companies, Market).

 

note : menu will appear depends on version of application 

Application Configuration

 

       Module function settings are based on configuration according to user requirements. Also, menu will be depends on version of application 

 

Steps

                            Open Parameter Teritory

1.        Loggin to Gen21 CMS

2.       Open Teritory Menu ( Menu-> Program Management -> Parameters -> Teritory )

                Add Teritory

                1. Open the Teritory window
                2. Press mouse right button for open the popup menu
                3. Press Edit in Popup menu
                4. Press Add in Popup menu
                5. Enter Name
                6. Press Save from Main menu
 

                Delete Source

                1.   Open the Teritory window
                2.   Press mouse right button for open the popup menu
                3.   Select the record in list
                4.   Press Edit in Popup menu
                5.   Press Delete in Popup menu 
                6.   Press Save from Main menu

                If Record is allready used by transactions, Deletion can not possible
