Manage Blank Media Budget

Description

Manage the budget for borrowing media / tape based on the department or program and this budget is given every month for 1 year

 

Application Configuration

Module function settings are based on configuration according to user requirements

Step

Open Manage Blank Media Budget

1.  Loggin to Gen21 CMS

2.  Open Module Manage Blank Media Budget  ( Media Library --> Manage Blank Media Budget ) 

 

Add Manage Blank Media Budget

1.  Press Right Button for open the popup menu

2.  Press edit in popup menu

3.  Press add in popup menu

4.  Select Field Mandatory

5.  Press save from main menu

 

Delete Manage Blank Media Budget

1.  Press Right Button for open the popup menu

2.  Press edit in popup menu

3.  Press add in popup menu

4.  Select data & click delete

5.  Press save from main menu

 

 

 

 

If the data has been used, it cannot be deleted

Note :

The process for filling in the departmental budget data and the program process is the same
