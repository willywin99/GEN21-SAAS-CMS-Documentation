Move existing program is moving a program from one PO to another PO, provided that the PO that you want to move still has a place for the program to be moved.

 

Open the Move Existing Program

1.    Login to Gen21CMS

2.    Open the Program Purchase Contract (Menu-> Program Management-> program Purchase Contract-> Program Info)

3.    In search Tab, Press Search Button

4.    In Browse tab, Select the PO and Open the Form Tab

5.    Press Button Move Existing Program
