License Type

 

Description
The type of license that will be used for the write-off process

 

 

Application Configuration

 

Module function settings are based on configuration according to user requirements. Also, menu will be enabled depends on version of application 

 

Steps

                            Open Parameter Source

1.       Loggin to Gen21 CMS

2.       Open License Type Menu ( Menu-> Program Management -> Parameters -> License Type )

                Add License Type

                1. Open the License Type window
                2. Press mouse right button for open the popup menu
                3. Press Edit in Popup menu
                4. Press Add in Popup menu
                5. Enter Name
                6. Press Save from Main menu
 

                Delete License Type

                1.   Open the License Type window
                2.   Press mouse right button for open the popup menu
                3.   Select the record in list
                4.   Press Edit in Popup menu
                5.   Press Delete in Popup menu 
                6.   Press Save from Main menu

                If Record is allready used by transactions, Deletion can not possible
