License Type

 

Description
The type of license that will be used for the write-off process

 

 

Application Configuration

 

Module function settings are based on configuration according to user requirements. Also, menu will be enabled depends on version of application 

 

Steps

                            Open Parameter Source

1.       Loggin to Gen21 CMS

2.       Open License Type Menu ( Program Management -> Parameters -> License Type )

                Add License Type

                1. Open the License Type menu.
                2. Navigate to Form Tab.
                3. Click Edit Button.
                4. Click Insert Button.
                5. Fill in the Form.
                6. Click Save Button.


 

                Delete License Type

                1.   Open the License Type menu.
                2.   On Browse Tab, select License Type which one will be deleted.
                3.   Navigate to Form Tab or double click on selected License Type.
                4.   Click Edit Button.
                5.   Click Delete Button.
                6.   Click OK to confirm the deletion.

                If Record is allready used by transactions, Deletion can not possible
