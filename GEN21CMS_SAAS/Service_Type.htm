Used to creating the Service Types definition. It will be used by Media Services interface while providing and storing all about services information in Media Library.

Open the Service Type

1. Login to Gen21CMS.

2. Open the Primary Subjects (Menu-> Media Library -> Parameters -> Service Type)

 

Steps Adding Service Type

1. Open Service Type

2. Create new Service Type,

      a. Right click on mouse button on the header field and choose

          “add row” to create new Service Type

b. Define the Service Code, Service Name, Service Cost, Program Cost

      c.  Right click on mouse button and choose “save”

 

Steps Editing Service Type

1. Open Service Type

2. Edit specific Service Type,

      a. Right click on mouse button and choose “edit mode”.

  b. Edit the Service Name, Service Cost, Program Cost

      c.  Right click on mouse button and choose “save”

 

Steps Deleting Service Type

1. Open Service Type

2. Delete Service Type,

      a. Right click on mouse button and choose “edit mode”

      b. Right click on mouse button and choose “delete”
