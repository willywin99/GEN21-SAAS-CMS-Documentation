Used to Creating the 'Archive Keywords' definition. These keywords will label stored media to describe its contents at a more detailed level in order to facilitate quicker access to the right media and its contents. Each media can be labeled with a maximum of 15 (fifteen) Archive Keywords (as shown in the Receive Program Media , in the 'assign archive keywords' button section).

Open the Archieve Keywords

1. Login to Gen21CMS.

2. Open the Primary Subjects (Media Library > Parameters > Archieve Keywords). 

 

 

Steps Adding Archieve Keywords

1. Open Archieve Keywords

2. Create new Archieve Keywords,

      a. Navigate to Form Tab and click Insert button to create new Archieve Keywords

      b. Define the Key Type and Description

      c.  Click Save button.

 

 

Steps Editing Archieve Keywords

1. Open Archieve Keywords

2. Edit specific Archieve Keywords,

      a. On Browse Tab, select the appropriate Archieve Keywords.

      b. Navigate to Form Tab or double click on selected Archieve Keywords then click Edit Button.

      c . Edit the Key Type and Description

      d. Click Save button.

 

 

Steps Deleting Archieve Keywords

1. Open Archieve Keywords

2. Delete Archieve Keywords,

      a. On Browse Tab, select the appropriate Archieve Keywords. 

      b. Navigate to Form Tab or double click on selected Archieve Keywords then click Edit Button.

      c. Click Delete Button. 

      d. Click OK to confirm the deletion.
