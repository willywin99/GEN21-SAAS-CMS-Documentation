Used for creating and maintenance of Program Type records. These are the types of (each 'Program Category' defined earlier) the purchased material falls under (e.g. Comedy, Horror). Normally used to define the nature of the 'content' of the material.

Steps for Program Type

  1. Login to GEN21 CMS.
  2. Open Program Type menu. (Program Management => Parameters => Program Type)

Add Program Type

  1. Open Program Type menu.
  2. Navigate to Form Tab.
  3. Click Insert button or CTRL + I.
  4. Fill in the Form.
  5. Click Save button or CTRL + S.

Delete Program Type

  1. Open Program Type menu.
  2. On Browse Tab, select Program Type which one will be deleted.
  3. Navigate to Form Tab or double click on selected Program Type.
  4. Click Delete button
  5. Click OK to confirm the deletion.

*If Record is already used by transactions, Deletion is not possible.
