Working in synchronization with Receive stock shoot , this function manages the circulation (borrow & return) of, specifically, Stock Shoot (Archive)material by appropriate users in theorganization.

     Note: Material must be 'received' before any Check-In/Out action can be initiated.

     Steps to proceed

Before we create / update stock shoot check in out, we must be input Receive B lank Media, Blank Media check in out and Recive stock shoot.

1. Click on the Media Library > Stock Shoot Check-In/Out  menu item. The starting Stock Shoot Check-In/Out screen is displayed:

As before, the function starts with a 'Search' screen to locate the media to be worked on. Enter any search criteria(s) in available fields in the left subwindow and click 'Search'button to commence the search.  Alternately, click 'Clear' button to erase any criterias already-entered in all fields.

2. The output of the search is a list of all records found by the search, shown in 2 possible formats depending on the selected Tab at the bottom of the left subwindow ('Grid' or 'Treeview') The default is on the 'Grid' form (see clicked tab under each display below). At this point one may also click 'Search' tab to perform another (different) search.

3. At the Grid form view, select 1 or more media to be worked on by clicking them individually in the leftmost 'X' column (or select/deselect ALL by clicking the 'X' column heading). Please observe items currently In or Out (colour indicated) while making your selections. The screen should now appear with the (set of) media already selected like this:

4.  Perform specific activities on the selected records by clicking any of the available functions listed as the horizontal yellow bars in the bottom of the right subwindow. Please refer to the following dedicated subsections for details on each chosen activity:

 

Cancel Last Transaction

 

            Process for cancel a program that has made a check out transaction. There are two cancel processes the first is cancel all, to cancel all programs that have been checked out with a note that the program you want to cancel has been  dragged and dropped on the left column, the second is cancel one, to cancel one of the programs that have been checked out with a note that the program you want to cancel has been dragged and dropped on the left column.

           

           Check Out

           The check out process is for the process of issuing a program that will be used for program borrowing. To cancel one of the programs that have been checked out with a note that the program you want to cancel has been dragged and dropped on the left column.

 

          

 

          Check In

 

          The check in process is for the process of returning the program that has been used. To check in one of the programs that have been checked out with a note that the program you want to cancel has been dragged and dropped on the left column.

 

          

 

          Media Properties

 

          Media Properties is one or more media distribution channels defined in the applicable Schedule.

 

         

 

          Historical Activities

 

          Historical activity is the storage of transactions that have been processed on the same day
