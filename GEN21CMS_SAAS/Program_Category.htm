Program Category

Description

Used for Creating and maintenance of Program Category records. These are the Categories the purchased material falls under (eg. Sports, Children Animation, General). Normally refering to the nature of the 'content' of the material.

Application Configuration

               Module function settings are based on configuration according to user requirements

Steps

                Open Parameter Program Category

1.        Loggin to Gen21 CMS

2          Open Program Category Menu ( Program Management -> Parameters -> Program Category )

 

                Add Program Category

                1. Open the Program Category window
                2. Press mouse right button for open the popup menu
                3. Press Edit in Popup menu
                4. Press Add in Popup menu
                5. Enter Name
                6. Press Save from Main menu


 

                Delete Program Category

                1.   Open the Program Category window
                2.   Press mouse right button for open the popup menu
                3.   Select the record in list
                4.   Press Edit in Popup menu
                5.   Press Delete in Popup menu 
                6.   Press Save from Main menu



                If Record is allready used by transactions, Deletion can not possible
