Affiliate channel valid for more than one channel.

               

Open the Channel Affilate

1.    Login to Gen21CMS

2.    Open the Program Purchase Contract (Menu-> Program Management-> program Purchase Contract)

3.    In search Tab, Press Search Button

4.    In Browse tab, Select the PO and Open the Form Tab

5.    Press Button Channel Affilate

Noted : Create new PO and When creating a PO source must select Affilate

Steps Adding New Channel Affilate

1. Open Channel Affilate

2. Create new Channel Affilate,

                a. Right click on mouse button on the header field and choose

                 “add row” to create new Channel Affilate

                b. Define the Code and Name

                c.  Right click on mouse button and choose “save”

 

Steps Editing Channel Affilate

1. Open Channel Affilate

2. Edit specific Channel Affilate,

                a. Right click on mouse button and choose “edit mode”.

                b. Edit the Code and Name

                c. Right click on mouse button and choose “save”

 

Steps Deleting Channel Affilate

1. Open Channel Affilate

2. Delete Channel Affilate,

                a. Right click on mouse button and choose “edit mode”

                b. Right click on mouse button and choose “delete”
