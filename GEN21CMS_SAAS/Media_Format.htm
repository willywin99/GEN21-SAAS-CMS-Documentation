Used to creating of Media Format definition. This is the variety of available format, brand and duration of the recording media, eg. Betacam 60", CD 30", DV cam 32" etc.

 

Open the Media Format

1. Login to Gen21CMS.

2. Open the Primary Subjects (Menu-> Media Library -> Parameters -> Media Format)

 

Steps Adding Media Format

1. Open Media Format

2. Create new Media Format,

      a. Right click on mouse button on the header field and choose

          “add row” to create new Media Format

b. Define the Code, Name and Digital File

      c.  Right click on mouse button and choose “save”

 

Steps Editing Media Format

1. Open Media Format

2. Edit specific Media Format,

      a. Right click on mouse button and choose “edit mode”.

      b. Edit the Name and Digital File

      c.  Right click on mouse button and choose “save”

 

Steps Deleting Media Format

1. Open Media Format

2. Delete Media Format,

      a. Right click on mouse button and choose “edit mode"

      b. Right click on mouse button and choose “delete”
