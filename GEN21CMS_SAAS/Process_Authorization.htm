Used for maintaining User Features Authorization(s) for each module/sub module in the system.

  1. User Feature Authorization(s) has 2 level of authorization module level. This is used to give access of the module to a user.
  2. Authorization within a module (Authorization Feature). This is used to give read, write, and delete access for each transition to a user.

Basic Setup for Process Authorization

Got Access Admin rights for EDP menu to GEN21 CMS.

Open the Process Authorization

  1. Login to GEN21 CMS.
  2. Open the Process Authorization menu. (Master = > Process Authorization)

Grant/Revoke Access to Module Level

  1. Open the Process Authorization.
  2. Open the Browse tab and select user.
  3. Open the Form tab.
  4. Select the module from the list of features.
  5. To grant the access (Read/Write/Delete) select the appropriated check box.
  6. To revoke the access (Read/Write/Delete) unselect the appropriated check box.
  7. Click Save from main menu.

Grant/Revoke Access to Particular Item (Authorization in module)

  1. Open the Process Authorization.
  2. Open the Browse tab and select user.
  3. Select the specific feature from Detailed Features.
  4. To grant the access (Read/Write/Delete) select the appropriated check box.
  5. To revoke the access (Read/Write/Delete) unselect the appropriated check box.
  6. Click Save from main menu.
