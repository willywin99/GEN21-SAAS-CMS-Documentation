To add, delete and reduce Runs in a Program.

 

Steps to proceed

1. Click on the  Program Management >  Program Purchase Contract > Program Info > Modify Run menu item.

2. To add Runs, click the 'Add Runs' radio button, enter the values in modifiable fields then click the OK button.

3. To delete Runs, click the 'Delete Runs' radio button, enter the values in modifiable fields then click the OK button.

4. To reduce Runs, click the 'Reduce Runs' radio button, enter the values in modifiable fields then click the OK button.
