Used to to create and maintain (view, add, modify or delete) information on Purchase transactions identified by  PO #. This section is mainly focusing on the 'Header' part of the PO ie. the 'Contract'. Please refer to the Program Info section for the 'Material Contents' of the PO.


Note : After the purchase contract has been completed, programs that have received NO PO can continue the Receive Media, Quality Control, Allocate and Posting Slot processes.

 

Steps to add new PO

  1. Open the program Purchase Contract
  2. Open the Form tab
  3. Press right click for open the popup menu
  4. Press edit in popup menu
  5. Press Add new in popup menu
  6. Enter the contract ref no
  7. Enter the PO date
  8. Select the major code ( local/import)
  9. Enter No of Programs
  10. Select Channel Name
  11. Enter Package Name
  12. Select Distributor name
  13. Select the original PH
  14. Select the Source
  15. Enter the No of Runs
  16. Select the Currency
  17. Press Save from main menu

 

Steps for Approve PO

1.Open the program Purchase Contract

2.Open the Form tab

3.In search Tab, Press Search Button

4.In Browse tab, Select the PO and Open the Form Tab

5.Press mouse right button for open the popup menu

6.Press edit in popup menu

7.Select the PO Approve Check box
