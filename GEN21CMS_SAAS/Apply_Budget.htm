Used to Apply Budget on the parameters that have been made in Manage Program Budgets

Open the Program Apply Budget 

1.    Login to Gen21CMS

2.    Open the Application Configuration (Menu-> Account -> Apply Budget).

Steps for add Assign Costing Budget  

1.   Open the window Apply Budget 

2.   Enter P.O.#  

3.   Enter Program Title

4.   Enter Episode#

5.   Enter Budget

6.   Press save from  Apply Selected Budget
